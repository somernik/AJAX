Clockpicker Plugin Example - Sarah Omernik

This plugin allows users to select hours and minutes for an input field using a clock style interface. Once a user clicks the input field a clock face will appear allowing the user to select the hour in military time. After hours have been selected, the user selects the minute value.

Some other features of this plugin are:

  • setting a default time for when the clock face first opens
  • having a done button or allowing autoclose once minute is selected
  • select the placement of the popup clock face (top, bottom, left, right)
  • selecting the popup arrow alignment (left, right)
