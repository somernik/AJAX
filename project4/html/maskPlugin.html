jquery Mask Plugin Example - Sarah Omernik

This plugin allows users to type only the necessary parts of an input such as numbers or letters and dynamically adds other characters like commas, dashes and parentheses.

This is very helpful you want to keep consistent input with the same format. If you need a user to input a phone number they would only have to type the numbers, 1234567890. The input is then formatted as defined in the JavaScript code, for example (123) 456-7890.

To test, simply type numbers into the input fields
